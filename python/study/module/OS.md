### OS

1. os.getcwd() : 현재 디렉토리 반환
2. os.chdir('디렉토리 경로') : 디렉토리 변경
3. os.listdir(): 디렉토리에 있는 파일,폴더 이름 리스트로 반환
4. os.list('디렉토리 경로') : 해당 경로에 있는 파일,폴더 이름 리스트형식으로 반환
5. os.path.join(해당경로, 파일이름) : 해당경로의 파일이름까지의 경로를 반환.
