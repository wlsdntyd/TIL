## Git 사용법



- ### 파일 업로드

  - **git** 연동 폴더에 파일 추가
  - **git status** 상태 확인
  - **git add .** 추가하기
  - **git commit -m "업로드 내용" **:파일 업로드
  - **git push origin master**

- ### 1. 파일 가져오기

  - **git clone [https://github](https://github/) 주소**

- ### 2. 파일 가져오기

  - **git pull origin master**
