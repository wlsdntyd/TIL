## 터미널 명령어들

- `ls`(폴더명): 폴더 내부의 파일 & 폴더를 나열
- `pwd(폴더명)`: 현재 폴더 전문 출력
- `mkdir`(폴더명): 폴더 생성
- `cd`(폴더명) change diretory 폴더 변경
- `rm`: rm 파일명.유형 => 파일 삭제
- `git status`: 상태 확인
- `touch`: touch 파일명.유형 => 파일 생성
