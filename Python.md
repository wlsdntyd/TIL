  - 딕셔너리

    - dust = {'영등포구' : 50, '강남구' : 40}

  - import random

    ```python
    import random
    coffee = ['아아', '뜨아', '라떼']
    co = random.choice(coffee)
    print(co)
    ```

    - 랜덤하게 '아아' or '뜨아' or '라떼' 중 하나 출력
